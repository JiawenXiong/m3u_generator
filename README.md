# M3U 播放列表生成器

一个简单易用的命令行工具，用于将指定文件夹中的视频/音频文件生成 M3U 播放列表，支持自定义重复次数和文件类型筛选。

## 功能特点

- 🎬 支持常见视频/音频格式（MP4, MKV, AVI, MOV, MP3, WAV）
- 🔄 自定义每个文件的重复播放次数
- 📂 可指定任意文件夹路径
- 🔧 可自定义文件扩展名筛选
- 📝 自动生成排序后的播放列表

## 安装要求

- Python 3.x

## 使用方法

### 基本语法

```bash
python m3u_gen.py <文件夹路径> [选项]
```

### 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `path` | 是 | 视频文件夹的路径 |
| `-n`, `--number` | 否 | 每个文件的播放次数（默认: 1） |
| `-e`, `--ext` | 否 | 指定文件后缀名（默认: .mp4 .mkv .avi .mov .mp3 .wav） |

### 使用示例

1. **最简用法**（每个文件播放 1 次）：

   ```bash
   python m3u_gen.py "D:/Videos/MyMovies"
   ```

2. **指定重复次数**（例如每个文件播放 5 次）：

   ```bash
   python m3u_gen.py "D:/Videos/Tutorials" -n 5
   ```

3. **指定特定后缀名**（只处理 .mp4 和 .ts 文件）：

   ```bash
   python m3u_gen.py "./videos" -n 3 -e .mp4 .ts
   ```

## 输出说明

程序会在指定的文件夹中生成名为 `playlist_x{n}.m3u` 的播放列表文件，其中 `{n}` 是重复次数。

示例输出：
```
成功生成播放列表: D:\Videos\MyMovies\playlist_x5.m3u
统计: 共有 10 个源文件，总计 50 个播放项。
```

## 生成的播放列表格式

生成的 M3U 文件包含以下内容：

- 文件头：`#EXTM3U`
- 文件路径列表（按文件名排序）
- 每个源文件重复指定的次数

## 注意事项

- 确保指定的文件夹路径有效
- 生成的播放列表使用绝对路径，便于在不同播放器中使用
- 文件按名称排序后加入播放列表
